-----------

DO_HOST C'est l'adresse IP de votre droplet sur DigitalOcean. Vous pouvez la trouver dans le tableau de bord de votre compte DigitalOcean, sous la section "Droplets". C'est l'adresse IP publique que DigitalOcean vous attribue lorsque vous créez un droplet.

DO_USERNAME C'est le nom d'utilisateur que vous utilisez pour vous connecter à votre droplet. Si vous avez choisi une image Ubuntu standard, alors le nom d'utilisateur par défaut est généralement root, sauf si vous avez configuré un autre utilisateur.

DO_PASSWORD C'est le mot de passe que vous utilisez pour vous connecter à votre droplet. Si vous avez choisi de configurer votre droplet avec un mot de passe lors de sa création, c'est ce mot de passe. Si vous avez choisi une clé SSH, cette option ne s'applique pas (mais dans votre cas, vous avez mentionné avoir choisi un mot de passe).
